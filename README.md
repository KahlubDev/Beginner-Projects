How to Run

Option 1: VS Code + CodeRunner (Recommended)
1. Open `happy_birthday.cpp`
2. Press `Ctrl + Alt + N` (or click the Play button)
3. Type the name and enjoy!

Option 2: Compile Manually
```

./happy_birthday          # Linux/macOS
# or
happy_birthday.exe        # Windows
```
